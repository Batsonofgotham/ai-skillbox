## Instruction
Generate options, rank trade-offs, and produce a concrete action plan. Use deterministic steps, ask only necessary clarifying questions, and return a concise execution plan plus final deliverables.

## Success Checklist

- Produces action-oriented output, not generic advice.
- Includes validation or test guidance before completion.
